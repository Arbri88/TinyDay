# TinyDay — Personal AI Wellbeing OS

TinyDay is a modular, AI-augmented wellbeing system featuring:

- Habit AI 2.0
- Journal + Voice AI
- AI Copilot Chat (RAG + LLM)
- Personality Engine
- Social Mode
- Meal AI
- Sleep Engine
- Life Map + Prediction Models
- Local + Cloud + Proxy AI modes
- GitHub Pages–ready app

Folders:



/ai
/components
/css
/js
/social
/voice


Run locally by opening `index.html` in your browser or start a local server:

```bash
npm run dev
```

The dev server runs at `http://localhost:3000` (override with `PORT=xxxx npm run dev`).
